# Village Defender - Official Website

A simple, responsive landing page for the Village Defender mobile game.

## About the Game

Village Defender is an exciting mobile strategy game where players defend their village against waves of enemies. Build defenses, upgrade your structures, and protect your village!

## Website Features

- Responsive design for all devices
- Game features showcase
- Gameplay instructions
- Direct link to Google Play Store

## Getting Started

Simply open `index.html` in a web browser to view the site locally.

## Download the Game

Get Village Defender on [Google Play](https://play.google.com/store/apps/details?id=com.eldemsoft.villagedefender&hl=tr)
